# 🎨 FinesseKart

**FinesseKart** is a full-stack online marketplace for paintings, where artists can showcase and sell their artwork, and users can explore, favorite, and purchase unique pieces. Built with modern web technologies, it delivers a seamless and interactive experience for both creators and buyers.

---

## 🖼️ Project Overview

FinesseKart connects **artists** and **art lovers** on a single platform:

* 🎨 Artists can **upload, showcase, and sell** their paintings
* 🛒 Users can **browse, favorite, and purchase** artwork
* 💡 Designed to promote creativity while making art accessible

---

## 🚀 Features

* 🔐 **User Authentication**
  Secure login and signup system using **JWT (JSON Web Tokens)**

* ❤️ **Favorites System**
  Users can save paintings to their favorites for later viewing

* 🛍️ **Marketplace Functionality**
  Browse, explore, and purchase paintings

* 👨‍🎨 **Artist Dashboard**
  Artists can manage and showcase their artwork

* 🌐 **Full Stack Application**
  Complete frontend and backend integration

---

## ⚙️ Tech Stack

* **Language:** JavaScript

* **MERN Stack:**

  * **MongoDB** – NoSQL database for storing users, paintings, and transactions
  * **Express.js** – Backend framework for handling APIs and server logic
  * **React.js** – Frontend library for building interactive UI
  * **Node.js** – Runtime environment for server-side development
